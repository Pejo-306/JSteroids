# JSteroids
NOTE: this is a temporary README file. It contains only instructions to 
launch the game in development mode.

# Run in a development environment
First, clone the project's GitHub repository and go to the project's root directory.

```bash
$ git clone https://github.com/Pejo-306/JSteroids.git
$ cd JSteroids/
```

Install all npm packages.

```bash
$ npm install
```

Run webpack's development server.

```bash
$ npm start
```

Afterwards, open your browser and visit **localhost:8000**.
